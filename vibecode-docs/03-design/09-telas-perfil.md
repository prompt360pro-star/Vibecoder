# VibeCode — Wireframes: Perfil

## Perfil Principal
- Avatar XL (96px) com gradient border + nível overlay
- Nome + username + bio
- Stats row: 🔥 Streak | ⚡ XP | 🏗️ Projetos (3 cards)
- Radar de Skills (6 eixos: Prompts, Frontend, Backend, Database, DevOps, UX)
- Progresso por nível (4 barras com %)
- Grid de conquistas (preview 8 + "Ver todas →")
- Botões: "VER PORTFOLIO" + "CERTIFICADOS"

## Conquistas
- Filter tabs: Todas (45) | Ganhas (12) | Próximas (5)
- Cards por categoria (Learning, Consistency, Building, AI, Social, Debug, Easter Eggs)
- Earned: colorido, data
- Locked: grayscale, opacity 0.4, progress bar se perto
- Near miss: glow border, "Faltam X!"

## Streak Detail
- Fire animation (Lottie) + número grande
- Record pessoal
- Calendário mensal (dias marcados com 🔥)
- Freeze tokens (visual tokens ❄️)
- Próximo bônus (card com progresso)
- Estatísticas: total dias ativos, melhor streak, freezes usados

## Settings
- Conta: Editar Perfil, Email, Senha, Refazer DNA Test
- Aprendizado: Lembrete, Meta diária, Idioma, Sons/haptics
- Notificações: Streak, Missões, Social, News (toggles)
- Assinatura: Plano atual + CTA upgrade
- Sobre: Termos, Privacidade, Avaliar, Contato, Versão
- Sair (destructive)
