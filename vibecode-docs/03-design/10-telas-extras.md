# VibeCode — Wireframes: Telas Extras

## Daily Challenge
### Disponível
- Ilustração animada + título
- Descrição do desafio
- Detalhes: dificuldade, tempo limite, XP, requisitos checklist
- Stats da comunidade (pessoas, tempo médio, recorde)
- Botão: "ACEITAR DESAFIO" (pulsing glow)
- Skip option (perde streak)

### Em Andamento
- Timer (counting up ou countdown)
- Requisitos checklist (atualiza em tempo real)
- Split view: Editor + Preview
- Action bar: Run, Vi, Undo, Submit

### Concluído
- Estrelas + tempo + posição no ranking
- Distribuição visual (histogram)
- Botões: Ver top 3, Compartilhar, Refazer, Voltar

## Scan & Learn
### Camera
- Camera viewfinder com focus area (dashed, pulsing purple)
- Capture button (72px, white ring)
- Gallery + Paste options

### Result
- Thumbnail da imagem scaneada
- Detection badge (Erro/Código/Docs)
- Vi response: explicação + solução + code
- Missão relacionada (link)
- Actions: Perguntar mais, Novo Scan

## Boss Fight
### Intro
- Cinematic animation (Vi em forma avançada)
- Título épico + gradient gold
- Skills usadas (badges)
- Recompensas listadas
- Botão: "INICIAR BOSS FIGHT" (gold gradient, glow)

### Em Andamento
- Boss HP bar (hearts ❤️🖤, gradient red→green)
- Etapa atual + instrução do Vi
- Checklist de sub-passos
- Editor de código
- Action bar: Run, Vi, Full Editor, Done

### Vitória
- Epic confetti + gold particles
- Vi evolui para nova forma
- Rewards card: XP, badge, certificado, próxima ilha
- Live URL do projeto
- Botões: Portfolio, Compartilhar, Próxima Ilha, Mapa

## Portfolio
- URL bar: portfolio.vibecode.app/username
- WebView do site gerado
- Template selector (horizontal scroll): Minimal, Modern, Bold, Hacker
- Actions: Trocar template, Compartilhar, PDF, LinkedIn

## Certificado
- Card com design profissional + gold border
- Dados: nome, nível, data, skills, projects, ID
- QR code de verificação
- Actions: Baixar PDF, LinkedIn, Compartilhar

## Paywall
- Vi PRO form (glow)
- Feature list com checkmarks verdes
- Pricing cards: Mensal ($14.99) vs Anual ($99 — "POPULAR")
- Lifetime option ($299)
- Garantia 7 dias
- Fine print: Termos, Privacidade, Restaurar
