# VibeCode — Wireframes: Social

## Feed
- Header: "COMUNIDADE" + notification bell
- Tabs: Feed | Ranking | Equipes | Prompts
- Posts cards com: avatar, nome, nível, tempo, conteúdo, code block, attachment, live link, actions (clap, comment, share)
- FAB: "✍️ PUBLICAR" (floating, gradient)

## Post Types
- Achievement: 🎉 + screenshot + celebração
- Prompt: 💡 + code block + copy button + tags
- Help: 🆘 + error code + "EU AJUDO!" button (yellow)
- General: 💬 + texto livre

## Ranking
- Period tabs: Semana | Mês | All Time
- Pódio visual: 🥇🥈🥉 (animated, particles for #1)
- Lista: posição, avatar, nome, XP, seta (↑↓═)
- Posição do aluno: highlighted, border purple

## Criar Post
- Nav: ✕ Cancelar + "NOVO POST" + Publicar
- Type selector: 4 opções horizontais (Feito, Prompt, Help, Geral)
- Text input multiline (h 120px)
- Code block input (font mono)
- Tags selector (chips)
- Attach buttons: Foto, Link, Projeto
- Live preview

## Prompts da Comunidade
- Search bar
- Category chips (horizontal scroll): Top, React, API, CSS, Debug, IA, SQL
- Prompt cards: título, autor, rating, saves, preview do prompt, copy/save/share
