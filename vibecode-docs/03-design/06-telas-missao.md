# VibeCode — Wireframes: Missão

## Mission Player
- Nav: ← Voltar + "MISSÃO X" + Vi button
- Progress bar de fases (fase X de Y)
- Conteúdo da fase atual (varia por tipo)
- Botões de ação na base

## Fase: Story
- Animação Lottie (h 200px)
- Bolha de narração do Vi
- Botão: "CONTINUAR"

## Fase: Concept
- Ilustração/diagrama interativo
- Texto explicativo
- Cards tocáveis para detalhes
- Código com partes coloridas (toque para explicar)
- Botão: "ENTENDI"

## Fase: Interaction
- Instrução clara
- Exercício interativo (drag-drop, quiz, V/F, etc)
- Feedback por item (verde/vermelho)
- Haptic feedback
- Botão: após completar

## Fase: Sandbox
- Instrução + dica
- Input de prompt para Vi
- Código gerado pelo Vi
- Preview visual do resultado
- Botão: "RODAR" + "CONTINUAR"

## Fase: Quiz
- Pergunta + código (se aplicável)
- 4 opções (A, B, C, D)
- Feedback ao selecionar (acertou/errou + explicação)

## Mission Complete
- Confetti animation
- Estrelas ganhas (1-3)
- XP popup animado
- Badge se desbloqueou
- Botões: "PRÓXIMA MISSÃO" / "VOLTAR AO MAPA"
