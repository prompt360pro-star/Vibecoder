# VibeCode — Wireframes: Home / Mapa

## Home Screen
- Status bar: 44px
- Header: Saudação + streak badge (h 56px)
- XP Bar: nível + barra de progresso
- Daily Challenge Card: ilustração + título + XP + tempo + botão
- Seção "Sua Jornada": título
- Island Map: scroll vertical com 4 ilhas
- Card "Continue": gradient card com missão atual
- Tab Bar: 5 tabs (Home ativo)

## Island Map
- 4 ilhas verticais conectadas por caminhos
- Cada ilha: emoji + nome + status (X/Y missões) + barra
- Ilha atual: colorida, acessível
- Ilhas futuras: escurecidas com cadeado
- Posição do aluno indicada: "← VOCÊ ESTÁ AQUI"
- Nuvens decorativas entre ilhas

## Ilha Expandida (ao tocar)
- Nav: ← Voltar ao Mapa
- Nome da ilha + progress bar total
- Lista de zonas com:
  - Nome da zona + emoji
  - Lista de missões:
    - ✅ completa (verde)
    - 🔵 atual (azul, borda destacada)
    - ⬜ pendente (cinza)
    - 🔒 bloqueada (cadeado)
  - Mini-projeto da zona
- Boss Fight no final (bloqueado até completar todas)
