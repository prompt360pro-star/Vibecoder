# Missão 08 — Terminal sem Medo

- ID: m08 | Zona: z02 Ferramentas | Tempo: 8min | XP: 40
- Fases: 5 (Story, Concept, Interactive Terminal, Sandbox, Quiz)

## Fase 1 — Story (40s)
Vi explorador abre porta para quarto escuro (terminal). Acende luz: painel amigável. "Terminal é só controle remoto do computador. Pedir pizza por telefone vs app."

## Fase 2 — Concept (90s)
Comparação visual: interface (clique) vs terminal (texto). Mesmo resultado. Por que devs usam terminal: rápido, poderoso, automatizável.

## Fase 3 — Interactive Terminal (120s)
Terminal simulado. 6 passos guiados: ls, cd projects, mkdir meu-app, ls, cd meu-app, cd ..
Vi guia cada passo com feedback.

## Fase 4 — Sandbox (60s)
Desafio livre: ls, criar pasta "portfolio", entrar, criar "images" dentro, voltar.
