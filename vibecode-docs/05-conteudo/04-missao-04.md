# Missão 04 — Gerando Código pela Primeira Vez

- ID: m04 | Zona: z01 | Tempo: 8min | XP: 40
- Fases: 5 (Story, Concept, Demo, Sandbox, Quiz)

## Fase 1 — Story (30s)
Vi como mágico. "Chegou a hora do show!"

## Fase 2 — Concept (60s)
3 ingredientes de uma página web: HTML (tijolos), CSS (pintura), JS (elétrica).

## Fase 3 — Demo (120s)
Vi cria cartão de visita digital ao vivo. Código aparece linha por linha. Preview renderizado. Tocar em partes do código para Vi explicar.

## Fase 4 — Sandbox (120s)
Aluno cria SEU cartão de visita pedindo ao Vi. Pode iterar ("quero mudar").

## Fase 5 — Quiz (45s)
"Qual é um prompt BOM para gerar código?" (4 opções)
