# Missão 05 — Lendo Código sem Pânico

- ID: m05 | Zona: z01 | Tempo: 8min | XP: 50
- Fases: 5 (Story, Concept, Interactive Code, Interaction, Quiz)

## Fase 1 — Story (40s)
Vi como detetive com lupa. "Se algo der errado, você precisa ler esse código."

## Fase 2 — Concept (90s)
- Anatomia HTML em cores (tag, atributo, valor, conteúdo) — tocar para explicar
- Anatomia CSS em cores (background, color, padding, border-radius, font-size) — preview muda ao tocar

## Fase 3 — Interactive Code (90s)
Laboratório de cores: mudar valores CSS com sliders/color pickers e ver preview em tempo real.

## Fase 4 — Interaction (60s)
2 perguntas: "O que esse código faz?" e "O que muda se trocar color:blue para green?"

## Fase 5 — Quiz (45s)
"Qual propriedade CSS arredonda os cantos?" border-radius✅
