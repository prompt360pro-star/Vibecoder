# VibeCode — PRD Fase 1: MVP (3 meses)

## Objetivo
Lançar app funcional com Ilha Básica completa, Vi funcional, gamificação e auth.

## Epics
1. E-1.1: Monorepo + Auth + Navegação (2 sem)
2. E-1.2: Onboarding + DNA Test (2 sem)
3. E-1.3: Vi Chat (backend + frontend) (3 sem)
4. E-1.4: Mission Player + 30 missões (4 sem)
5. E-1.5: Gamificação (XP, níveis, streaks, badges) (2 sem)
6. E-1.6: Perfil + Settings (1 sem)
7. E-1.7: Polish + QA + Deploy (2 sem)

## KPIs Target
- 10K downloads primeiro mês
- D7 retention: >50%
- NPS: >40
- Mission completion rate: >70%
- Crash rate: <1%

## Equipe: 5-7 pessoas
PM, Mobile Dev, Backend Dev, AI Engineer, Designer, Content Creator (0.5), QA (0.5)
