
# 📱 VIBECODE — O App Definitivo para Aprender Vibe Coding

---

## 🧠 FILOSOFIA DE DESIGN

```
Se EU fosse aprender vibe coding do zero,
o que me faria VICIAR em aprender?

Resposta: Eu quero SENTIR que estou construindo algo REAL
desde o minuto 1. Não quero slides. Não quero PDFs.
Quero um mentor pessoal de IA no meu bolso
que me conheça, me desafie e me faça
construir apps reais enquanto aprendo.
```

---

## 🎯 CONCEITO CENTRAL

> **"Aprenda a construir apps com IA construindo apps com IA."**
> O app ENSINA vibe coding USANDO vibe coding como método.
> É meta. É prático. É viciante.

---

## 📛 IDENTIDADE

```
Nome:        VIBECODE
Tagline:     "Code the future. Ride the vibe."
Slogan PT:   "Construa o futuro. Surfe na vibe."

Mascote:     "Vi" — um robô-mentor simpático e adaptativo
             que evolui visualmente conforme o aluno avança.

             Nível Básico:    Vi é um cubo luminoso simples 🟦
             Nível Médio:     Vi ganha braços e ferramentas 🤖
             Nível Avançado:  Vi vira um robô completo ⚡
             Nível Expert:    Vi se transforma em holograma 🌟

Cores:       Gradiente Roxo→Azul→Cyan (futuro, IA, tecnologia)
             Dark mode padrão (como os devs gostam)
             
Tipografia:  JetBrains Mono (código)
             Inter (UI)
```

**Perguntas do DNA Test:**

```
1.  Qual sua relação com tecnologia?
2.  Você já programou algo? (nem que seja HTML)
3.  Quando vê código, o que sente?
4.  Já usou ChatGPT/Claude para algo?
5.  O que quer CONSTRUIR? (escolha múltipla com ícones):
    - 📱 App mobile
    - 🌐 Site/Landing page
    - 🛒 Loja online
    - 🤖 Chatbot/IA
    - 📊 Dashboard
    - 💰 SaaS (Software como Serviço)
    - 🎮 Jogo
    - 🔧 Automatização
6.  Como você aprende melhor?
    - 📖 Lendo
    - 🎥 Assistindo
    - 🔨 Fazendo
    - 🗣️ Conversando
7.  Quanto tempo por dia pode dedicar?
    - ⚡ 10 min (micro-doses)
    - ☕ 30 min (café learning)
    - 🔥 1 hora (focado)
    - 🚀 2h+ (modo intenso)
8.  Qual seu objetivo principal?
    - 💼 Mudar de carreira
    - 💰 Renda extra/freelance
    - 🧠 Criar meu próprio produto
    - 📚 Curiosidade/hobby
9.  Você tem computador? (Mac/Windows/Linux/Chromebook)
10. Em que área trabalha hoje?
```

## 🏗️ ARQUITETURA TÉCNICA DO APP

### Stack Tecnológica

```
📱 MOBILE APP:
├── Framework: React Native + Expo
├── Navegação: Expo Router
├── Estado: Zustand + React Query
├── UI: Tamagui + Custom Components
├── Animações: Reanimated 3 + Lottie
├── Editor código: Monaco Editor (web view) / CodeMirror
├── Storage local: MMKV + SQLite (Expo)
└── Push: Expo Notifications

🖥️ BACKEND:
├── API: Next.js API Routes (Edge Runtime)
├── Auth: Clerk
├── Database: PostgreSQL (Neon)
├── ORM: Drizzle
├── Cache: Redis (Upstash)
├── File Storage: S3 (Cloudflare R2)
├── Search: Meilisearch
└── Background Jobs: Inngest

🤖 AI LAYER:
├── Primary LLM: Claude 4 (Anthropic)
├── Secondary LLM: GPT-4o (OpenAI)
├── Fast LLM: Groq (Llama)
├── Embeddings: OpenAI text-embedding-3
├── Vector DB: Pinecone
├── AI Framework: Vercel AI SDK
├── Prompt Management: Custom (versioned)
├── Code Execution: Sandpack (client-side)
├── Observability: Langfuse
└── Guardrails: Custom middleware

🚀 INFRA:
├── Hosting: Vercel (API) + Fly.io (workers)
├── CDN: CloudFlare
├── CI/CD: GitHub Actions
├── Monitoring: Sentry + Better Uptime
├── Analytics: PostHog
├── Payments: Stripe
├── Email: Resend
└── Feature Flags: Unleash
```
