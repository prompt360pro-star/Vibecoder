# VibeCode — Sistema de Gamificação

## 1. XP (Experience Points)
- Missão: 30-100 XP
- Mini-projeto: 200-500 XP
- Boss Fight: 500-1000 XP
- Desafio diário: 50-150 XP
- Streak bonus 7d: 200 XP, 30d: 1000 XP
- Ajudar alguém: 50 XP
- Compartilhar prompt: 30 XP
- Badge: 100-500 XP

## 3. Streaks
- Dia ativo = streak incrementa
- Dia inativo sem freeze = reset
- Free: 2 freezes, Pro: 5 freezes
- +1 freeze a cada 7 dias de streak
- Bônus: 3d(50XP), 7d(200XP), 14d(300XP), 30d(1000XP), 60d(2000XP), 100d(5000XP), 365d(20000XP)

## 4. Conquistas (45 total)
### Aprendizado: Primeira Lição, Estudioso(10), Maratonista(50), Acadêmico(100), Perfeito, Enciclopédia
### Consistência: Centelha(3d), Chama(7d), Fogueira(14d), Inferno(30d), Vulcão(60d), Sol(100d), Supernova(365d)
### Construção: Primeiro Tijolo, Minha Casa(3), Arranha-céu(10), Cidade(all), Ship It!, Na Internet!, Ka-ching!
### IA: Primeiro Contato, Prompt Sniper, Prompt Wizard, Tool Master, AI Whisperer, Singularidade
### Social: Amigável, Mentor(10), Influencer, Equipe(3), Líder(top10)
### Debug: Bug Hunter, Detetive(10), Blindado, Performance Freak
### Easter Eggs: Night Owl, Early Bird, Konami, 42, Unicórnio

## 5. Boss Fights
- Básico: Landing Page (10 etapas, 500 XP)
- Intermediário: CRUD Fullstack (15 etapas, 800 XP)
- Avançado: Micro-SaaS (15 etapas, 1000 XP)
- Expert: Produto em Escala (20 etapas, 1500 XP)

## 6. Daily Challenges
- 1/dia, 3 dificuldades rotativas
- Fácil: 5min/50XP, Médio: 10min/100XP, Difícil: 15min/150XP
- Ranking global por tempo
